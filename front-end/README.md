# Setup Front-End Web com React, Vite, TypeScript e Tailwind CSS 4

## Passo a Passo

Acesse as pastas 

```bash
cd teste-foco
cd front-end
``

instale as dependências do projeto

```bash
npm install
``

Após instalar as dependências, inicie o projeto com o comando:

```bash
npm run dev
``

Acesse o projeto
[http://localhost:5173](http://localhost:5173)
